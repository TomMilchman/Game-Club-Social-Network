Read Me

Social Network Name:

Game Club

Additional Pages Added:

I added a page that would let the user know of upcoming big game releases and a page that contains a bunch of gaming trivia.

Additional Features Added:

Users have the ability to unlike posts and view how many followers each user has on their profile page.

What Was Hard To Do?

The authentication system was easily the most difficult and most time consuming part of the project for me.

Did I Have a Partner?

No. All of the work was done by myself (both server and client side). My ID is: 209371483.

Server Routes Supported:

  • /login (PUT)

  • /signup (POST)

  • /logout (PUT)

  • /search (GET)

  • /privileges (GET)

  • /feed (GET)

  • /users:

    • /users/:username/follow (PUT)

    • /users/:username/unfollow (PUT)

    • /users/:username/followinfo (GET)

    • /users/:username/posts (GET)

    • /users/following (GET)

    • /users/nonadmin (GET)

    /posts:

    • /posts/createpost (POST)

    • /posts/:username/like/:postid (PUT)

    • /posts/:username/unlike/:postid (PUT)

    /admin:

    • /admin/loginactivity (GET)

    • /admin/deleteuser/:username (DELETE)

    • /admin/enable/gamingtrivia (PUT)

    • /admin/disable/gamingtrivia (PUT)

    • /admin/enable/upcomingreleases (PUT)

    • /admin/disable/upcomingreleases (PUT)

    • /admin/enable/unlike (PUT)

    • /admin/disable/unlike (PUT)

    • /admin/enable/numoffollowers (PUT)

    • /admin/disable/numoffollowers (PUT)

Additional important info:

How to run the website: Run the server with "node server.js" and run the client with "npm start dev". Then go to http://localhost:5173 in your browser. To run the tests, run "node test.js".
